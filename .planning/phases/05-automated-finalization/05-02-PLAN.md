         ```
         def auto_analyze(
             ticker: str,
             *,
             config: AppConfig,
             out_dir: Path,
             analyst_client: ClaudeAnalystClient | None = None,
             validator_client: RedTeamValidatorClient | None = None,
             epistemic_client: EpistemicReviewerClient | None = None,
             sector_knowledge: dict | None = None,
             max_retries: int = 2,
         ) -> AutoAnalyzeResult:
         ```

       Orchestration flow:
       a. Fetch raw bundles: call run_live_scan([ticker], out_dir=out_dir, stop_at="raw-bundle", config=config)
       b. Load merged bundle from written_paths["merged_raw"] (JSON load)
       c. If sector_knowledge is None, try load_sector_knowledge from ticker's sector (catch exceptions, warn, proceed without)
       d. Create analyst client if not injected: ClaudeAnalystClient(config.anthropic_api_key, model=config.analyst_model)
       e. Retry loop (max_retries + 1 attempts):
          - Call generate_llm_analysis_draft(merged_bundle, client=analyst_client, sector_knowledge=sector_knowledge, validator_objections=objections)
          - Extract first structured_candidate from draft
          - Extract raw_candidate from merged_bundle["raw_candidates"][0]
          - Call validate_overlay(overlay_candidate, raw_candidate, client=validator_client)
          - If verdict == "APPROVE": break
          - Else: objections = validation["objections"], increment retries_used
       f. Run epistemic review:
          - extract_epistemic_input(overlay_candidate) to get EpistemicReviewInput
          - epistemic_review(review_input, client=epistemic_client) to get PCS answers
          - Merge epistemic PCS answers back into overlay_candidate's epistemic_inputs (replace analyst's entries with reviewer's for q1-q5 keys: answer, justification, evidence_source)
       g. Finalize:
          - Call finalize_structured_analysis(draft, reviewer=f"llm:{config.analyst_model}", final_status="LLM_CONFIRMED", note="Automated finalization...")
       h. Return AutoAnalyzeResult with all collected data
