# Step 5: Valuation

Keep the model simple and auditable.

## Base Equation

`Value = Revenue x FCF Margin x FCF Multiple`

Then adjust for:
- Net debt / cash
- Expected share count change

## Required Inputs

1. Revenue path
- Industry baseline + company initiatives + catalyst impact

2. FCF margin path
- Historical range + cost actions + operating leverage

3. FCF multiple
- Industry baseline adjusted for quality, growth, leverage, risk

4. Share count
- Flat unless clear buyback/dilution trend

## Return Gate

Default hurdle for new capital:
- 30%+ CAGR

Documented exception:
- 20%+ CAGR over 6+ years with top-tier operator and high durability

## Downside Model

Estimate a reasonable worst case:
- Non-catastrophic but adverse outcome
- Explicit downside from current price

## Codex Extrapolation

Require 3 outputs, not 1:
- Bear value
- Base value
- Stretch value

Decision relies on **probability-weighted asymmetry**, not base case alone.
