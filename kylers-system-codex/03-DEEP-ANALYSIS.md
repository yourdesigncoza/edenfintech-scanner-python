# Step 3: Deep Analysis (Relative Quality)

At this stage, analysis is comparative, not absolute.

## Compare Within Theme

For each peer:
- Leverage and liquidity resilience
- Margin structure and direction
- ROIC/ROCE through full cycle
- Growth quality (organic vs acquisition/dilution)
- Exposure to the key catalyst

## Priority Order

1. Survival runway / balance sheet quality
2. Margin durability or superior niche
3. Catalyst capture ability
4. Optionality and upside spread

## Keep / Cut Logic

Cut if clearly inferior and no compensating upside.

Keep as backup only if all are true:
- No hidden failure from earlier steps
- Materially higher upside than winner
- Pipeline lacks better alternatives

## Codex Extrapolation

Create a one-line decision memo per company:
- "Why this is better than peer A"
- "Why this is safer than peer B"
- "What can make me wrong quickly"

If you cannot write these three lines, work is incomplete.
